# Training Notebooks

Fine-tuning notebooks for all Vera models, designed for Google Colab with GPU acceleration.

## Prerequisites

- Google Colab account (free tier works, Pro recommended for faster training)
- HuggingFace account with Write-access token — see [HUGGINGFACE_SETUP.md](HUGGINGFACE_SETUP.md)
- Google Drive mounted (for saving checkpoints and loading data)

## Notebooks

| Notebook | Model | Task | GPU Required |
|----------|-------|------|-------------|
| `vera_decomposer_finetune_cot.ipynb` | `vera-decomposer-cot` | CoT claim decomposition with reasoning | A100 recommended |
| `vera_decomposer_finetune_non_cot.ipynb` | `vera-decomposer` | Standard claim decomposition | A100 recommended |
| `vera_verifier_finetune_cot.ipynb` | `vera-verifier-cot` | CoT verification with reasoning | A100 recommended |
| `vera_verifier_finetune.ipynb` | `vera-verifier-cot` | Compatibility copy of the same CoT verifier workflow | A100 recommended |

Notes:

- The standard production verifier is **not** a fine-tuned notebook in this repo; it uses zero-shot [ModernBERT-large-zeroshot-v2.0](https://huggingface.co/MoritzLaurer/ModernBERT-large-zeroshot-v2.0) directly in deployment.
- `vera_verifier_finetune_cot.ipynb` is the canonical verifier training notebook for the released CoT verifier artifact.
- `vera_verifier_finetune.ipynb` is retained because earlier project materials referenced that filename.

## Workflow

1. Upload the training data from `datasets/` to your Google Drive
2. Open the notebook in Google Colab
3. Set your HuggingFace token in the authentication cell
4. Run all cells in order
5. The trained model will be pushed to your HuggingFace account

## Expected Output

Each notebook produces:
- LoRA adapter weights saved to Google Drive
- Merged GGUF model (for llama.cpp deployment)
- Training loss curves and eval metrics
- Model pushed to HuggingFace Hub (if token configured)

## Training Data

The notebooks load data from `datasets/`:
- **Decomposer notebooks**: Use `datasets/distilled_cot/` or `datasets/distilled_non_cot/`
- **Verifier notebooks**: Use `datasets/distilled_verification/`

The decomposition datasets are generated by:

- `uv run python scripts/2_distill_decomposition.py --mode cot`
- `uv run python scripts/2_distill_decomposition.py --mode standard`

See [DATASETS.md](../DATASETS.md) for full details.
